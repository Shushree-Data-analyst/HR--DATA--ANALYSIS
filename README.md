📊 HR Data Analysis Project

📌 Project Overview

This project focuses on analyzing employee attrition and workforce trends using multiple tools like Excel, Python, SQL, Power BI, and Tableau.

---

📊 Key KPIs

- Attrition Rate
- Total Employees
- Average Age
- Attrition Count

---

📈 Visualizations

- Department-wise Attrition
- Age Group Analysis
- Job Satisfaction Rating
- Education Field-wise Attrition
- Gender-wise Attrition Rate

---

🛠 Tools & Technologies

- Excel
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI
- Tableau

---

🔍 Key Insights

- Identified departments with highest attrition
- Found age groups with higher employee turnover
- Analyzed job satisfaction impact on attrition
- Compared gender-based attrition trends

---

📂 Project Files

- Excel Dashboard
- Python Analysis Notebook
- Power BI Dashboard
- Tableau Dashboard
- SQL Queries

---

🚀 Conclusion

This project helps organizations understand employee behavior and reduce attrition through data-driven insights.
